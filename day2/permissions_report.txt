umask (после 027):файл rw-r---/директория rwx-x---
----------------------------------------------------
Значение 640/750 прав доступа
6 - первая цифра отвечает за права пользователя (user), обозначает rw - чтение и запис
4 - отвечает за права группы, r-только чтение
0 - Others - отстальные, отсутствие прав

750 
7 - пользователю доступны все права
5 - чтение + выполнение 
0 - отсутствие прав

- 640 = u:rw-, g:r--, o:---
- 750 = u:rwx, g:r-x, o:---
-------------------------------------------------------
UMASK БАЗОВЫЕ ПРАВА
для файлов макс права 666  rw-rw-rw
Максимальные права директории: 777  (rwxrwxrwx)
umask	Результат для папки	Объяснение
000	777	всё разрешено
022	755	стандарт по умолчанию (владельцу всё, остальным чтение+вход)
027	750	владельцу всё, группе чтение+вход, остальным — ничего
077	700	только владелец


tar:
- c/x/t = create/extract/list
- z = gzip, f = файл архива
- пример: tar -czf app-logs.tar.gz configs logs
- исключения: --exclude='*.log'
